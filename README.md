# Toebeans

A game project, early scaffolding stage — nothing playable yet, just the
foundation to build on.

## Running it

```
npm install
npm run dev
```

Then open the URL it prints (usually `http://localhost:5173`). You should
see a dark scene with an orange box drifting across a floor — that's the
placeholder "cat," proof the pipeline works end to end.

Other useful commands:

- `npm run check` — makes sure nothing is broken (types + tests). Run this
  after any change.
- `npm run build` — builds the game for production/sharing.
